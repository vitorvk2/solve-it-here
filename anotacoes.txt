Para criar um LINK simbólico entre a pasta interna de '/storage/app/public' com 'public/storage' basta dar o comando

$ php artisan storage:link

Esse link permitirá acessar a pasta mais interna de forma publica



Autenticação das Requsições:
    $ composer require tymon/jwt-auth:*
    Documentação: https://jwt-auth.readthedocs.io/en/develop/laravel-installation/


Instalação do Vue
    $ composer require laravel/ui:^3.2.1
    $ php artisan ui vue --auth
    $ npm install && npm run dev
    $ npm install vue-loader@^15.9.7 --save-dev --legacy-peer-deps
    $ npm install && npm run dev

Instalação do Redis
    sudo yum install epel-release
    sudo yum install snapd
    sudo systemctl enable --now snapd.socket
    sudo ln -s /var/lib/snapd/snap /snap
    sudo snap install redis

Instalação do DebugBar
    composer require barryvdh/laravel-debugbar
    'Debugbar' => Barryvdh\Debugbar\Facades::class [config]


Instalação do pacote Predis, responsável por fazer a conversa entre o php e Redis
    composer require predis/predis
